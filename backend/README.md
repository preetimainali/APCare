# Backend — Hospital Command Center

FastAPI API, data ingest, alert evaluation, and ML forecasting.

- `app/api/` — Route modules
- `app/core/` — Config, security, dependencies
- `app/models/` — Pydantic and DB models
- `app/services/` — Business logic
- `data/` — SQLite, seed data, pipelines
- `ml/` — Forecasting train/predict
